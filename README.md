# Avrae Frontend - Starter

This is a minimal Next.js 14 App Router frontend scaffold for Avrae.

How to use:
1. Fill NEXT_PUBLIC_API_URL in Vercel or .env.local for local dev.
2. Deploy to Vercel (import GitHub repo).

Pages:
- / : Landing with cinematic intro
- /apply : Application form (posts to API)
- /thank-you : Post-submission confirmation
